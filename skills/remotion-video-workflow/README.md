# remotion-video-workflow

Primary installable skill package for `infocard-video-skills`.

## Purpose

This skill turns a content brief or source material into a structured Remotion short-video workflow.
It focuses on the workflow your team has already validated:

- Hook / Content / End Card structure
- storyboard-first planning
- cover and video linkage
- ratio-aware output
- render and verification

## Upstream Dependency

Use this skill together with `remotion-best-practices`.

Recommended split:

- `remotion-best-practices` for Remotion primitives and implementation rules
- `remotion-video-workflow` for this repository's video production workflow
