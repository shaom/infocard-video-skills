## Suggested Title

`v0.1.0 - Repository foundation`

## Suggested Release Notes

Initial public foundation for `infocard-video-skills`.

This release establishes the repository as a standalone Remotion video capability repo for agents, separate from the information-card workflow repository.

Included in this release:

- one primary skill package: `remotion-video-workflow`
- workflow references for structure, ratios, branding, audio, cover linkage, render, and quality checks
- lightweight helper scripts for render and output inspection
- repository-level documentation for positioning, publishing, and contribution guidance
- GitHub collaboration templates and security policy

Current scope:

- repository structure
- skill contract
- workflow documentation
- publishing-ready project layout

Not included yet:

- a full example Remotion project
- CI tied to a runnable example
- additional installable skills

## Suggested Short Release Summary

First public repository foundation for reusable Remotion video skills.
